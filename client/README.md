# Client reference for Vizo Glass (Raspberry Pi / test client)

CEO e criador: David Adriano Ferrari dos Santos

Este diretório contém scripts de referência para testar o backend local do Vizo Glass.

client/text_client.py -- enviar mensagens de texto ao backend
client/capture_and_send.py -- (esqueleto) captura de áudio e envio via WebSocket (não implementado em produção aqui)

Ajuste BACKEND_URL em client/config.py antes de executar.
